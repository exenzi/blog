## –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç

–ü–æ-—Å—É—Ç–∏ —ç–Ω—É–º–µ—Ä–µ–π—Ç - —ç—Ç–æ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä, –∫–æ—Ç–æ—Ä—ã–π –æ—Ç–¥–∞—ë—Ç –Ω–∞–º –∏–Ω–¥–µ–∫—Å –∏ —ç–ª–µ–º–µ–Ω—Ç. –¢–æ –µ—Å—Ç—å –¥–æ–±–∏—Ç—å—Å—è —Ç–æ–≥–æ –∂–µ —ç—Ñ—Ñ–µ–∫—Ç–∞ –º–æ–∂–Ω–æ —Ç–∞–∫:

```python
def enumerate(sequence, start=0):
    n = start
    for elem in sequence:
        yield n, elem
        n += 1
```
